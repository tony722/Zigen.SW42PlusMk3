# Zigen HXL-44Plus / HXL-88Plus Crestron Module
### Zigen.HxlPlus

This module is written in SIMPL# with SIMPL+ wrapper for use in SIMPL Windows. Use in the same way you'd use a Crestron Digital Media switcher.
